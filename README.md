# Simple Full-stack Task Tracking App

A simple task tracking application built with **Hono** and **Vue**.  
The project is intended for learning, experimentation, and exploring a lightweight full-stack setup.

## Tech stack

-   **Backend:** Hono
-   **Frontend:** Vue
-   **Database:** MySQL

## Getting started

Install dependencies:

```bash
npm install
```

Run the project in development mode:

```bash
npm run dev
```

Run the project with enhanced (colored) logs:

```bash
npm run dev:colored
```

## Notes

This project is not production-ready and is used primarily for educational purposes.
